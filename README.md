
# Movie App Flutter

E-commerce platform for Vietnamese shoppers to purchase goods from the US with Flutter
Link website: https://govietmart.com/

# Architecture: Clean Architecture

![image](https://github.com/trunghau121/movie_app_flutter/assets/14820422/90d64a31-b2fb-4e77-863a-91b3eb6d1e0a)

- Domain: Entities, UseCases, Abstract Repositories
- Data: Rest API(dio), Models
- Presentation: Screens, Widgets, Blocs + Cubits(bloc_flutter)
- DI: GetIt
